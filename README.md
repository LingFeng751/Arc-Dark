## ArcaeaDarkMode项目介绍
深色模式的Arcaea，以助于在夜晚游玩Arcaea，降低视觉疲劳

## 适用人群
* 被散光等视力问题困扰，无法看清光侧铺面  
* 喜欢晚上关灯玩，但觉得主界面和光侧谱面晃眼睛  
* 推分时，被光侧谱面可读性差的问题所困扰
