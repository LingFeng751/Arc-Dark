## ArcaeaDarkMode項目介紹
深色模式的Arcaea，以助於在夜晚遊玩Arcaea，降低視覺疲勞

## 適用人群
* 被散光等視力問題睏擾，無法看清光側鋪麵  
* 喜歡晚上關燈玩，但覺得主界麵和光側譜麵晃眼睛  
* 推分時，被光側譜麵可讀性差的問題所睏擾
